# react-carousel
An app that shows a slideshow of multiple pictures.
# Usage
```
npm run watch
```
# Use
* HTML5
* CSS3
* React
* React-DOM
* Babel
* Babelify
* Watchify
* Browserify
* Modern JavaScript (ES2015+)
* MongoDB
* Express
* Node.js
* Nodemon
